About

SocialiteProviders is a collection of OAuth 1 & 2 packages that extend Laravel Socialite.

The Observer Pattern is used in the Manager package to extend Socialite. This allows numerous providers to be used in addition to the ones provided by Laravel Socialite (they can be overridden easily). Only the providers that you add to the listeners array will be used.

Nearly all of the providers where created by @DraperStudio while the Manager for the providers was created by @AndyWendt.
